# 硬件引脚分配

## radio

SPI接口管理两个cc1101芯片，通过不同的选择信号，实现对两个芯片的分时控制；

| pin(stm8s208RB) | port         | pin(cc1101) | note |
| --------------- | ------------ | ----------- | ---- |
| 38              | PC5/SPI_SCK  | SCK         | 芯片A  |
| 41              | PC6/SPI_MOSI | MOSI        | 芯片A  |
| 42              | PC7/SPI_MISO | GDO1        | 芯片A  |
| 16              | PF4          | CSN         | 芯片A  |
| 15              | PF5          | GDO0        | 芯片A  |
| 14              | PF6          | GDO2        | 芯片A  |
| 38              | PC5/SPI_SCK  | SCK         | 芯片A  |
| 41              | PC6/SPI_MOSI | MOSI        | 芯片A  |
| 42              | PC7/SPI_MISO | GDO1        | 芯片A  |
| 13              | PF7          | CSN         | 芯片A  |
| 9               | PA3          | GDO0        | 芯片A  |
| 10              | PA4          | GDO2        | 芯片A  |
